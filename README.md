# 🌍 Earth's Heartbeat



---

## 📖 About

**Earth's Heartbeat** is a lightweight Android application designed to bring together Schumann Resonance monitoring stations from multiple locations into one convenient app.


Whether you're interested in Earth's natural electromagnetic activity, space weather correlations, or simply enjoy monitoring resonance data, Earth's Heartbeat provides a simple and easy-to-use interface.

---

## ✨ Features

- 🌍 View Schumann Resonance charts from multiple stations
- 🇷🇺 Tomsk Resonance Data
- 🇮🇹 Cumiana Resonance Data
- 🔄 Automatic hourly data updates
- 📱 Android home screen widgets
- 🎯 Choose your preferred resonance station for widgets
- ⚡ Fast and lightweight
- 🎨 Clean and simple user interface
- 📊 Easy access to multiple resonance sources in one place

---

## 📱 Android Widgets

Earth's Heartbeat includes customizable Android widgets.

### Widget Features

- Display your favorite Schumann Resonance station
- Automatically updates with new data
- Multiple widget instances supported
- Quick viewing without opening the app

### Adding a Widget

1. Long-press your Android home screen
2. Tap **Widgets**
3. Locate **Earth's Heartbeat**
4. Drag the widget onto your home screen
5. Select the resonance station you wish to display
